<p style="font-family: SimSun; font-size: 14px; color: rgb(0, 0, 0); margin: 10px 0px; padding: 0px; list-style: none; line-height: 26px; background-color: rgb(254, 254, 254); overflow: visible !important;">美国东部时间12月21日晚间消息，随着财政悬崖的最后期限迫近，《华盛顿邮报》的资深编辑斯蒂夫-莫夫松提议，在美国政府不需增税和削减支出的情况下，最有效的解决方式就是卖掉阿拉斯加。他指出，中国财力充足，是美国最大的债权国，或将成为最有可能的买家。</p>

<p style="font-family: SimSun; font-size: 14px; color: rgb(0, 0, 0); margin: 10px 0px; padding: 0px; list-style: none; line-height: 26px; background-color: rgb(254, 254, 254); overflow: visible !important;">　　近日，全球资本市场都随着美国&ldquo;财政悬崖&rdquo;的谈判进展而发生波动起伏，两党的唇枪舌战耗时良久已经让美国经济遭受损害。目前美国政府的债务高达16万亿美元，如果民主党与共和党在年底之前无法就财政方案达成一致意见，美国经济将在明年初摔下高达6000亿美元的&ldquo;财政悬崖&rdquo;。国际三大信用评级机构之一的惠誉也在19日发出警告，美国或因此丧失3A主权评级。</p>

<p style="font-family: SimSun; font-size: 14px; color: rgb(0, 0, 0); margin: 10px 0px; padding: 0px; list-style: none; line-height: 26px; background-color: rgb(254, 254, 254); overflow: visible !important;">　　日前，《华盛顿邮报》资深编辑斯蒂文-莫夫松发表文章，为美国解决&ldquo;财政悬崖&rdquo;指出一条看似两全其美的方法&mdash;卖掉阿拉斯加州。莫夫松表示，这个想法既不疯狂也不荒谬，因为没什么东西能比16万亿美元的债务更荒谬。而中国目前现金充足，是美国最大的债主，并且正面临住地缺乏的局面，把阿拉斯加卖给中国或许是一个好办法。</p>

<p style="font-family: SimSun; font-size: 14px; color: rgb(0, 0, 0); margin: 10px 0px; padding: 0px; list-style: none; line-height: 26px; background-color: rgb(254, 254, 254); overflow: visible !important;">　　莫夫松在文章中指出，中国现在拥有大量现金、但缺乏能源和住地。既然如此，中国在投资上为什么一定要忍受到美国外国投资委员会对国家安全的质疑呢？比如在收购加拿大Nexen公司或者美国电池制造商A123系列时遇到的不公待遇。而购买阿拉斯加可就容易多了！别忘了，中国还有1.1615万亿美元的外汇储备投资了美国国债，现在是美国国债最大的持有者。通过这笔交易，中美之间可以简单地取消债务，从而避免了中国一旦在公开市场出售这些债券而造成的金融动荡。</p>

<p style="font-family: SimSun; font-size: 14px; color: rgb(0, 0, 0); margin: 10px 0px; padding: 0px; list-style: none; line-height: 26px; background-color: rgb(254, 254, 254); overflow: visible !important;">　　此外，买下阿拉斯加，有助于解决目前中国人满为患、严重污染的城市问题。随着北极冰层的融化和气候的变化，阿拉斯加也可以作为一条有价值的航线，让中国到欧洲的游轮或商业船只往来更节省时间。</p>

<p style="font-family: SimSun; font-size: 14px; color: rgb(0, 0, 0); margin: 10px 0px; padding: 0px; list-style: none; line-height: 26px; background-color: rgb(254, 254, 254); overflow: visible !important;">　　同时，莫夫松也认为，俄罗斯或许会凭借其历史的渊源抢先在中国之前&ldquo;收复失地&rdquo;。在美国买下阿拉斯加之前，俄罗斯已经在这块土地上管辖了126年。鉴于普京正在寻求机会恢复俄罗斯的权力和荣耀，购买阿拉斯加给了普京一个重建前俄罗斯首都锡特卡小镇的机会。</p>

<p style="font-family: SimSun; font-size: 14px; color: rgb(0, 0, 0); margin: 10px 0px; padding: 0px; list-style: none; line-height: 26px; background-color: rgb(254, 254, 254); overflow: visible !important;">　　文章指出，一些私人买家如美国&ldquo;地产之王&rdquo;唐纳德-特朗普以及中东石油富豪可能也会对此项交易有兴趣。对爱出风头的特朗普来说，他可以在北美最高峰麦金利山上做广告，或重新命名它为&ldquo;特朗普山&rdquo;。而坐拥大型主权财富投资基金的中东石油出口国，可能愿意在地球正在变暖的形势下为自己找个&ldquo;冷静&rdquo;的空间。更重要的是，如果一个OPEC中的国家购买了阿拉斯加，将会推动OPEC在世界石油市场的份额，并提升该组织的定价能力。</p>

<p style="font-family: SimSun; font-size: 14px; color: rgb(0, 0, 0); margin: 10px 0px; padding: 0px; list-style: none; line-height: 26px; background-color: rgb(254, 254, 254); overflow: visible !important;">　　在欧美经济危机蔓延了三年之后，卖州卖岛已经不再是什么新鲜事。此前，希腊总理就表示，只要出售岛屿对国家安全不构成威胁，希腊的一些岛屿就可以用作商业开发。两年前，背负着1.18万亿英镑国债的意大利，也曾经大规模出售了包括多座岛屿、沙滩、王宫和城堡等国家财产。</p>
